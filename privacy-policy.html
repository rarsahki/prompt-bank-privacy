Privacy Policy for Prompt Bank & Enhancer Extension

Last updated: Ocotober 2025

Quick Summary: This extension stores your API key locally, processes your text prompts through Google's Gemini API, and does not collect, store, or transmit any personal data to our servers. Your data stays on your device.

1. Information We Collect

1.1 Data Stored Locally on Your Device

This extension stores the following information locally on your device using Chrome's storage APIs:

  • Gemini API Key: Your Google Gemini API key (stored securely in Chrome's sync storage)
  • Extension Settings: Your preferences and configuration options (stored in Chrome sync storage)
  • Visibility Control Settings: Your visibility preferences (AI Websites only, This Page, All Pages) stored in Chrome sync storage
  • Enabled Domains List: List of domains where you've enabled "This Page" functionality (stored in Chrome sync storage)
  • Error History: Local error logs for debugging purposes (limited to last 50 entries, stored in Chrome sync storage)
  • UI State: Temporary state information for floating button and API call progress (stored in browser localStorage)

1.2 Data Processed (Not Stored)

The extension processes the following data but does not store it:

  • Text Prompts: The text you type in input fields that you choose to enhance
  • Enhanced Text: AI-generated enhanced versions of your prompts
  • Website Information: Basic site detection for UI customization (site name, favicon)

2. How We Use Your Information

2.1 Prompt Enhancement

When you use the enhancement feature:

  • Your text prompt is sent to Google's Gemini API for processing
  • The enhanced result is displayed to you in the extension interface
  • No data is stored on our servers or transmitted to third parties (except Google's API)

2.2 Visibility Control

The extension uses your visibility preferences to determine where to show the floating button:

  • AI Websites Only: Shows the button only on AI-related websites (ChatGPT, Gemini, Claude, etc.)
  • This Page: Allows you to enable the button on specific domains you choose
  • All Pages: Shows the button on all websites
  • Your domain preferences are stored locally and synced across your devices (if Chrome sync is enabled)
  • No domain information is transmitted to our servers

2.3 Error Handling

When errors occur:

  • Error information is stored locally for debugging and retry functionality
  • Error history is limited to 50 entries and stored only on your device
  • No error data is transmitted to our servers

3. Data Sharing and Third-Party Services

3.1 Google Gemini API

This extension uses Google's Gemini API for text enhancement. When you use the enhancement feature:

  • Your prompt text is sent to Google's servers
  • Google's privacy policy applies to this data transmission
  • We do not control how Google processes this data
  • You can review Google's privacy policy at: https://policies.google.com/privacy

3.2 No Data Collection

We do not:

  • Collect personal information
  • Track your browsing activity
  • Store your prompts or enhanced text
  • Share data with advertisers or other third parties
  • Use analytics or tracking services

4. Data Security

4.1 Storage Security

Your data is stored using two Chrome storage methods:

  • Chrome Sync Storage (chrome.storage.sync): Used for API keys, settings, and error history - syncs across your devices with encryption
  • Browser localStorage: Used for temporary UI state management (floating button state, API call progress) - local only, not synced

Both storage methods provide:

  • Encryption at rest
  • Access controls managed by Chrome
  • Sync across devices (for sync storage only, if Chrome sync is enabled)

4.2 API Key Security

Your Gemini API key is:

  • Stored securely in Chrome's sync storage (chrome.storage.sync)
  • Not transmitted to our servers
  • Only used to authenticate with Google's API
  • Syncs across your devices if Chrome sync is enabled

5. Your Rights and Controls

5.1 Data Access and Deletion

You can:

  • View your stored settings in the extension popup
  • Change your visibility control preferences at any time
  • Add or remove domains from your "This Page" enabled list
  • Clear your API key and settings at any time
  • Clear error history from the extension interface
  • Uninstall the extension to remove all stored data

5.2 API Key Management

You control your Gemini API key:

  • You provide your own API key from Google AI Studio
  • You can change or remove your API key at any time
  • We never see or store your API key on our servers

6. Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us.

7. Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify you of any changes by:

  • Posting the new privacy policy on this page
  • Updating the "Last updated" date
  • Notifying you through the extension if significant changes are made

8. Data Retention

We retain data as follows:

  • Local Settings: Retained until you clear them or uninstall the extension
  • Visibility Control Settings: Retained until you change them or uninstall the extension
  • Enabled Domains List: Retained until you remove domains or uninstall the extension
  • Error History: Limited to 50 entries, older entries are automatically deleted
  • API Keys: Retained until you remove them or uninstall the extension
  • Prompts and Enhanced Text: Not stored, processed in real-time only

9. International Data Transfers

When you use the Gemini API, your data may be processed by Google in various countries. Google's data processing is subject to their privacy policy and applicable data protection laws.
